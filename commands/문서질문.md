# 단일 문서 Q&A

입력: **$ARGUMENTS**

## 동작

1. `$ARGUMENTS`에서 첫 공백 앞을 파일 경로, 이후를 질의로 분리.
   - 파일 경로에 공백이 있으면 따옴표 필수 (`"path with spaces" 질문`).
2. 파일 존재 확인. 없으면 오류 보고.
3. `Agent(subagent_type="document-qa-agent")` 호출:
   - 입력: `file_path`, `question`
4. 답변을 그대로 표시.

## 출력

document-qa-agent가 반환한 답변 + 근거 인용 + 누락 정보 고지.

## 다중 문서 질의 안내

질의가 여러 파일을 가로질러야 한다면(예: "계약서와 판결문을 비교하면?"), 사용자에게 `/사건질문 <사건폴더> <질의>`를 권유.
