# 사건 폴더 통합 Q&A

입력: **$ARGUMENTS**

## 동작

1. `$ARGUMENTS`에서 사건 폴더 경로와 질의 분리.
   - 폴더 경로 예: `notes/법무/사건/2026가단12345-홍길동v김철수`
2. 폴더 존재 + `사건개요.md` 존재 확인.
3. `Agent(subagent_type="case-folder-qa-agent")` 호출:
   - 입력: `case_folder`, `question`
4. RAG 스타일 답변을 표시 (파일별 근거 + 타임라인 + 누락 정보).

## 활용 시나리오

- "이 사건에서 계약 해제 통지가 언제 어떻게 이루어졌나?"
- "상대방 서면 주장의 변화 과정을 정리해달라."
- "손해액 주장의 근거 자료 목록을 뽑아달라."
- "의뢰인 진술과 계약서 내용이 모순되는 지점이 있는가?"

## 주의

- 질의가 법령·판례 해석에 관한 것이면 `/리서치`로 분기.
- 단일 파일에만 해당되면 `/문서질문`이 더 정확.
- 답변 분량이 커지면 에이전트가 자동으로 상위 10개 파일로 축약 + 추가 질의 권유.
