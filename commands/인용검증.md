# 인용 일괄 검증

대상: **$ARGUMENTS**

## 동작

1. 파일 존재 확인.
2. `Agent(subagent_type="citation-checker-kr")` 호출:
   - 입력: 파일 경로
3. 검증 보고서 표시.
4. 사용자 선택:
   - **[ ] 경미 이슈만 자동 적용** — 형식 오류·표기 통일만 Edit.
   - **[ ] 전체 적용** — 중대 이슈 수정안도 Edit.
   - **[ ] 수동 검토** — 적용 없이 보고서만.

## 활용 시나리오

- 서면 제출 직전 최종 검증 (`/초안` 파이프라인에 이미 포함됨 — 독립 실행은 외부에서 받은 서면 검토 용도)
- 다른 사무실에서 받은 준비서면의 인용 실존 여부 확인
- 과거 자문서·의견서의 인용 재검증

## 실패 처리

- verify_citations MCP 장애 시: 개별 `get_law_text`/`get_decision_text`로 백오프 검증.
- 인용이 너무 많아(50건+) 일괄 호출이 실패하면 배치 분할 실행.
