# 법률 문서 요약

대상: **$ARGUMENTS**

## 동작

1. 인자 파싱: 파일 경로 + 선택적 깊이 옵션(`--tldr` / `--상세`; 기본 표준).
2. 파일 유형 자동 식별 (판결문/계약서/서면/조서/진술서).
   - PDF는 `anthropic-skills:pdf` 스킬을 통해 텍스트 추출 후 요약.
   - DOCX는 `anthropic-skills:docx` 스킬 활용.
3. `Agent(subagent_type="legal-summarizer-kr")` 호출:
   - 입력: 파일 경로, 깊이, 유형(추정치)
4. 요약 결과를 **stdout에 표시** + 선택적으로 파일로 저장 (`{원본경로}.summary.md`).

## 출력

에이전트가 반환한 구조화 요약을 그대로 표시.

## 후속 제안

- "세부 질문이 있다면 `/문서질문 {파일} {질문}`으로."
- "사건 폴더 일부라면 `/사건질문 {폴더} ...`로 통합 조회."
- "계약서라면 `/계약검토 {파일}`로 리스크 감사."
