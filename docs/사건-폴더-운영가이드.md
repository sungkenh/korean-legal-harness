# 사건 폴더 운영 가이드

슈퍼로이어의 "사건 기반 대화" 기능을 파일시스템으로 구현한 구조입니다. 사건별 자료를 한 폴더에 모아두면 `/사건질문`, `/초안` 커맨드가 폴더 전체를 참조합니다.

## 표준 구조

```
notes/법무/사건/{사건번호}-{당사자약칭}/
├── 사건개요.md          # 당사자·청구취지·쟁점 요약 (최초 작성 필수)
├── 관련법령.md          # 쟁점별 법령·판례 정리 (리서처 결과 누적)
├── 증거목록.md          # 갑·을 제N호증 목록 + 입증취지
├── 일정.md              # 기일·제출기한
├── 서면/                # 준비서면·답변서·고소장 등 작성 폴더
│   ├── 01-답변서.md
│   └── 02-준비서면(1).md
├── 자료/                # 원본 PDF·HWP·DOCX·이미지
│   ├── 계약서.pdf
│   ├── 판결문-1심.pdf
│   └── 녹취록.docx
└── 통신/                # 의뢰인·상대방·법원 연락 기록
    └── 2026-04-22-의뢰인상담.md
```

## 폴더명 규칙

- 형식: `{사건번호}-{당사자약칭}` (예: `2026가단12345-홍길동v김철수`)
- 의뢰단계 사건(번호 미부여): `의뢰-{YYYYMMDD}-{약칭}` (예: `의뢰-20260422-김고객이혼`)
- 한글·영문·숫자·하이픈만. 공백·특수문자 금지.

## 사건개요.md 템플릿

```markdown
# 사건개요

## 기본정보
- 사건번호: 2026가단12345
- 법원·재판부: 서울중앙지방법원 민사제1단독
- 담당변호사: OOO
- 사건분류: 민사 / 손해배상
- 의뢰일: 2026-04-22

## 당사자
- 원고: 홍길동 (의뢰인)
- 피고: 김철수
- 대리인: 법무법인 OOO

## 청구취지·신청취지
1. 피고는 원고에게 금 100,000,000원 및 이에 대한 ...

## 사실관계 요약
(시간순 3~5문단)

## 쟁점
1. 계약 해제의 유효성
2. 손해배상 범위

## 현황·진행상황
- 2026-04-22 소장 접수
- 2026-05-15 답변서 제출 예정

## 관련 파일
- 자료/계약서.pdf
- 자료/해제통지서.pdf
```

## 워크플로우

### 1) 사건 생성
```
/사건생성 2026가단12345 홍길동v김철수
```
→ 폴더 자동 생성 + `사건개요.md`·`관련법령.md`·`증거목록.md` 템플릿 삽입.

### 2) 자료 수집
- 의뢰인 제공 문서를 `자료/`에 배치.
- 스캔 PDF는 그대로 두면 됨 (`anthropic-skills:pdf` 스킬이 OCR 처리).
- HWP 파일은 DOCX 또는 PDF로 변환 후 배치(한글 네이티브 파싱은 미지원).

### 3) 통합 질의 (사건 기반 대화)
```
/사건질문 notes/법무/사건/2026가단12345-홍길동v김철수 계약 해제 통지는 언제 어떤 방식으로 이루어졌나?
```
→ `case-folder-qa-agent`가 폴더 내 모든 파일을 Glob·Grep으로 훑어 근거 인용과 함께 답변.

### 4) 법령 리서치 축적
```
/리서치 계약해제 후 손해배상 범위
```
→ 결과를 `관련법령.md`에 **추가** (덮어쓰지 않음).

### 5) 서면 초안
```
/초안 답변서 2026가단12345
```
→ `legal-longform-drafter`가 사건개요·관련법령·증거목록·자료를 읽고 `서면/01-답변서.md` 작성.

### 6) 인용 검증
```
/인용검증 notes/법무/사건/2026가단12345-홍길동v김철수/서면/01-답변서.md
```

## 원칙

- **원본 보존** — `자료/`의 원본은 이름 변경·편집 금지. 작업본은 별도 파일로.
- **타임라인 유지** — `통신/`은 날짜 접두사(`YYYY-MM-DD`)로 시간순 정렬.
- **증거번호 일관성** — 갑·을 + 한글 숫자 + 숫자(예: `갑제1호증`). 국어 순서 준수.
- **개인정보 분리** — 주민번호·계좌 등은 별지로. 본 폴더의 `개인정보.md`(암호화 필요 시 외부)로 분리.
