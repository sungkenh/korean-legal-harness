# 계약서 검토 체크리스트

## 공통 기본 항목

- [ ] 당사자 표시 (법인명·대표자·주소) 정확성
- [ ] 계약의 목적·범위 명확성
- [ ] 계약기간·갱신 조건
- [ ] 대가·지급조건·지연이자
- [ ] 의무·책임의 범위
- [ ] 손해배상 한도·예정액
- [ ] 해제·해지 사유
- [ ] 비밀유지(NDA)·경업금지
- [ ] 지식재산권 귀속
- [ ] 준거법·관할 법원 (전속관할 여부)
- [ ] 분쟁해결(중재 vs 소송)
- [ ] 통지 방법 (서면·이메일)
- [ ] 불가항력(Force Majeure)
- [ ] 완전합의 조항·수정 방식

## 리스크 등급

| 등급 | 의미 | 대응 |
|---|---|---|
| 🔴 High | 당사자에게 중대한 불이익 | 반드시 수정 요구 |
| 🟡 Medium | 협상 가능한 불균형 | 수정 시도, 불가 시 보완조항 |
| 🟢 Low | 표준 조항 | 확인 후 통과 |

## 유형별 추가 체크

### 매매계약
- 하자담보책임 기간
- 소유권 이전 시점
- 위험부담 (민법 제537조·제538조)

### 용역·도급
- 완성물 검수 기준
- 수정 요청 횟수·비용
- 재위탁 금지 여부

### 근로계약
- 근로기준법 제17조 필수기재사항 (임금·근로시간·휴일·휴가)
- 수습기간·해고예고
- 비밀유지·경업금지 (퇴사 후 기간·지역·보상)

### 라이선스
- 독점/비독점
- 지역·기간·분야 제한
- 로열티 산정·감사권

### M&A·투자
- 진술 및 보장 (Reps & Warranties)
- 면책 조항 (Indemnification)
- Material Adverse Change (MAC)
- 우선매수권·Tag-along·Drag-along

## 금지·무효 조항 경고

- 소비자기본법·약관규제법 위반 (부당 면책·부당 손해배상)
- 하도급법 위반 (부당한 대금 감액)
- 공정거래법 위반 (거래상 지위 남용)
- 민법 제103조 위반 (반사회질서)
