# 인용 형식 규칙

## 법령

### 기본
- `「민법」 제750조` — 법명은 낫표(「 」) 안에.
- `「민법」 제750조 제1항` — 항은 "제1항".
- `「민법」 제750조 제1항 제2호` — 호는 "제2호".
- `「상법」 제389조 제3항, 제209조 제2항` — 준용 조문.

### 시행령·시행규칙

- `「도로교통법 시행규칙」 제3조 제1항 [별표 1]`

### 개정 이력 명시 필요 시
- `「민법」(2023. 5. 16. 법률 제19409호로 개정된 것) 제844조`
- **시행일 기준**으로 적용 법령 확정. 개정 전 사건은 **구법** 명시.

## 판례

### 표준 형식
`대법원 2023. 5. 11. 선고 2022다123456 판결`

구성:
- 법원명: 대법원 / 서울고등법원 / 서울중앙지방법원 등
- 선고일: YYYY. M. D. (연월일 사이 점 + 공백)
- "선고" (결정은 "자")
- 사건번호: 2022다123456 (연도 + 사건부호 + 번호)
- 종류: 판결 / 결정 / 명령

### 결정·명령
- `대법원 2024. 3. 15.자 2023마1234 결정`
- `헌법재판소 2023. 7. 20. 선고 2022헌바123 결정`

### 판례집 병기 (선택)
- `대법원 2022. 1. 27. 선고 2021다234567 판결 [공2022상, 456]`

### 전원합의체
- `대법원 2023. 10. 26. 선고 2022다12345 전원합의체 판결`
