# 의뢰인 보고서·법률자문서 양식

## 1. 사건진행보고서 (의뢰인 정기 보고용)

```markdown
# 사건진행보고서

- 사건: [사건번호 및 사건명]
- 의뢰인: [성명]
- 담당변호사: [성명]
- 보고일: YYYY-MM-DD

## 1. 진행 경과
(직전 보고 이후 발생한 사실·기일·제출 서면 등을 시간순)

## 2. 현재 상황
- 다음 기일: YYYY-MM-DD HH:MM, [법원·재판부]
- 상대방 최근 주장:
- 당방 대응 방향:

## 3. 쟁점 및 전망
(쟁점 2~4개 + 각 쟁점의 현재 유불리 평가)

## 4. 의뢰인께 요청사항
- [ ] 추가 자료 제출
- [ ] 면담 일정 조율
- [ ] 합의 의사 확인

## 5. 다음 보고 예정일
YYYY-MM-DD
```

## 2. 법률자문서 (단일 쟁점 자문)

```markdown
# 법률자문서

- 의뢰인: [법인명·대표자]
- 자문 요청일: YYYY-MM-DD
- 작성일: YYYY-MM-DD
- 작성자: 변호사 OOO

## I. 자문 요청사항
(의뢰인이 제기한 질의 원문)

## II. 사실관계
(의뢰인 제공 자료 및 면담에 기초하여 정리)

## III. 쟁점
1. 쟁점 1
2. 쟁점 2

## IV. 관련 법령
- 「법명」 제N조 제N항
- 「시행령」 제N조

## V. 관련 판례
- 대법원 YYYY. M. D. 선고 사건번호 판결 — 판시사항 요지
- (유사 사실관계 판례 2~4건)

## VI. 검토 의견
### 1. 쟁점 1에 관하여
(법령 적용 → 판례 태도 → 본 사안 적용 → 결론)

### 2. 쟁점 2에 관하여
(동일 구조)

## VII. 결론 및 권고
(실무적 대응 방안 3~5개 번호 항목)

## VIII. 한계 및 유의사항
- 본 자문서는 제공된 자료에 기초하며, 새로운 사실 발견 시 결론이 변경될 수 있습니다.
- 최종 판단은 담당 변호사와의 협의를 거쳐 주시기 바랍니다.
```

## 3. 의견서 (법원·수사기관 제출용)

- 표제: "변호인 의견서" / "참고인 의견서" 등 용도 명시
- 사건번호·당사자 표시
- 의견의 요지 (1문단 요약)
- 사실관계
- 법리
- 결론
- 입증방법 목록

## 공통 원칙

- **결론 우선** — 바쁜 의뢰인·재판부를 위해 첫 섹션에 결론 요약.
- **불확실성 명시** — "~할 가능성이 있다", "~로 판단될 여지가 있다" 등 확정 단언 회피.
- **근거 모두 인용** — 법령 조문과 판례 사건번호 100% 명시.
- **A4 기준 분량** — 사건진행보고서 1~2매, 법률자문서 3~8매, 의견서 사안별.
- **부록 분리** — 긴 판례 원문·법령 조문은 말미 부록으로.
