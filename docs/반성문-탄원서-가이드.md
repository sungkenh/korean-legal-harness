# 반성문·탄원서 작성 가이드

## 목적

형사 양형자료. 재판부에 **진지한 반성**과 **재범 방지 의지**를 전달한다. 감형 요소로 활용된다.

## 반성문 표준 구성

1. **사건 경위 인정** — 범행 사실을 미화·축소하지 않고 솔직히 기재.
2. **피해자에 대한 사과** — 피해 회복 노력(합의·공탁·치료비 지원 등) 구체적 서술.
3. **범행 동기 반성** — "왜 그랬는지"를 핑계 없이 진술. 환경 탓 지양.
4. **재범 방지 계획** — 치료·상담·환경 변화 등 실행 가능한 조치.
5. **가족·사회적 책임** — 부양가족, 직업, 사회복귀 의지.
6. **선처 호소** — 겸손한 어조. "선처를 바란다"는 표현 1회로 족함.

## 탄원서 (제3자 작성)

- 탄원인 인적사항 + 피고인과의 관계
- 피고인의 평소 인성·생활
- 사건의 정상참작 사유
- 선처 호소

## 작성 톤

- **1인칭 진솔체** — "저는", "제가". 문어체보다 구어체에 가깝게.
- **짧은 문장** — 한 문장 20자 내외 권장.
- **구체적 사실** — "깊이 반성하고 있습니다" → "매일 피해자께 사죄 편지를 쓰고 있습니다".
- **금지어** — "억울하다", "실수였다", "기억이 나지 않는다" (반성 부정 신호).

## 분량

- 반성문: A4 1~2매
- 탄원서: A4 1매 (다수 제출 시 각 1매)

## 개인정보 처리

주민등록번호 뒷자리, 연락처, 계좌번호는 **마스킹** 또는 **별지** 처리. 저장 시 훅이 자동 경고.

## 법적 효과

- 형법 제51조(양형의 조건) 중 "범행 후의 정황"에 해당
- 형사소송규칙 제155조에 따른 양형자료
- **합의·공탁과 병행** 시 효과 증대
